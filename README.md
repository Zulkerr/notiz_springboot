# Notizen-App mit Spring Boot

Dies ist eine einfache Webanwendung zum Verwalten von Notizen mit Kategorien. Sie wurde mit **Spring Boot**, **Thymeleaf**, **Bootstra** und einer **H2-In-Memory-Datenbank** entwickelt.
